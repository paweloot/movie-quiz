# Movie Quiz
Android quiz application testing one's knowledge about celebrities, movie soundtracks and movie clips. 

Currently **in development**.

## Built using
  * MVVM Architecture
  * Kotlin
  * Jetpack Navigation Component
  * Data Binding
  * Glide
  * ExoPlayer
